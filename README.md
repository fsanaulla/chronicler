## Chronicler
Open-source [Scala](https://www.scala-lang.org/) toolchain for [InfluxDB](https://www.influxdata.com/).

## Features
- Multiple backend
- Flexible API
- Code generation with macros
- High modularity
- Spark support (located [here](https://github.com/fsanaulla/chronicler-spark))
- Easy to customize
